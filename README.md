# 自己肯定感ブースター (200の問い #011)

「できたこと」を再認識し自己肯定感を育てる

## 特徴 (δ方式)

- API呼び出しゼロ・コストゼロ・永久動作
- ChatGPT / Claude / Gemini のどれでも使える
- 5つの問いに答えるだけで、最適化された分析プロンプトを自動生成
- 履歴保存、他者視点プロンプト、深掘りプロンプト機能付き

## 開発

```bash
npm install
npm run dev
```

## デプロイ

Vercelに接続するだけで自動デプロイされます。
